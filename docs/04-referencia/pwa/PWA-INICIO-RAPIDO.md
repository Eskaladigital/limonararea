# 🚀 Inicio Rápido - PWA Administrador

## ¿Qué es esto?

Ahora el panel de administrador de Eco Area Limonar se puede **instalar como una app en tu móvil**.

## 📱 Instalar en tu Móvil

### iPhone (Safari)
1. Abre Safari
2. Ve a: `https://www.ecoarealimonar.com/administrator`
3. Inicia sesión
4. Toca el botón de **Compartir** (⬆️)
5. Desplázate y toca **"Añadir a pantalla de inicio"**
6. Toca **"Añadir"**
7. ¡Listo! Verás el icono de Eco Area Limonar Admin en tu pantalla

### Android (Chrome)
1. Abre Chrome
2. Ve a: `https://www.ecoarealimonar.com/administrator`
3. Inicia sesión
4. Aparecerá un banner: **"Instalar app"**
   - Si no aparece: Menú (⋮) → "Instalar aplicación"
5. Toca **"Instalar"**
6. ¡Listo! Verás el icono de Eco Area Limonar Admin

## ✨ Ventajas

Una vez instalada:
- ✅ Se abre como una app normal (sin barra del navegador)
- ✅ Más rápida (guarda datos en caché)
- ✅ Funciona mejor sin conexión
- ✅ Accesos directos a Reservas, Vehículos y Clientes

## 🔧 ¿No aparece el banner?

Si no ves el banner de instalación:
1. Asegúrate de estar usando HTTPS
2. No estés en modo incógnito
3. Espera 5 segundos después de cargar la página
4. Si lo cerraste, vuelve a aparecer en 7 días

Para forzar la instalación:
- **iOS Safari**: Botón Compartir → "Añadir a inicio"
- **Android Chrome**: Menú → "Instalar aplicación"

## ❓ Preguntas Frecuentes

**¿Necesito instalarla?**
No, es opcional. Pero mejora mucho la experiencia en móvil.

**¿Ocupa mucho espacio?**
No, es muy ligera (menos de 5MB).

**¿Funciona offline?**
Parcialmente. Guarda páginas visitadas para verlas sin conexión.

**¿Cómo la desinstalo?**
- iOS: Mantén presionado el icono → "Eliminar app"
- Android: Ajustes → Apps → Eco Area Limonar Admin → Desinstalar
